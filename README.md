# Maze
A python maze game in terminal.

## Level File Format
First line: `@width,height`, for exmple: `@10,10`

#### Symbol
- ` ` for road
- `$` for wall
- **`P`** layer
- **`G`** ate
- **`K`** ey
- **`E`** nd
