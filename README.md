# Maze
A python maze game in terminal.

## Level File Format
At the header of the file:
```
@size=(10,10)
```

#### Symbol
- ` ` for road
- `$` for wall
- **`P`** layer
- **`G`** ate
- **`K`** ey
- **`E`** nd
